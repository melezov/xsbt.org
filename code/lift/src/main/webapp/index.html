Legend:

  • Version:
    Standardized launcher name, which may differ from the original filename
    (e.g. all 0.11.x are originally named just "sbt-launch.jar")
  • Date:
    Date when the launcher was first published on the binary repository.
  • Size:
    Size in kibibytes.
  • Remote:
    Direct URL link to the remote HTTP binary repository.
  • Proxy:
    This link performs a redirect to the official binary release repository.
    Essentially this is an URL shortening service at the moment, but the idea is that if the original repo goes down the proxy link will redirect to the next available failover repository or serve the local launcher instance.
  • Local:
    Local copy of the release, with
    direct HTTP downloads from http://xsbt.org/ and
    direct HTTPS downloads from https://xsbt.org/
  • FTP:
    Local copy of the release, direct FTP download.
    FTP download index is available at ftp://xsbt.org/
    Alternatively, you can also use the FTPs (secure) connection at ftps://xsbt.org/
  • Gopher:
    Local copy of the release, direct Gopher download.
    Gopher download index is available at gopher://xsbt.org/
  • SHA-1:
    Local copy of the SHA-1 launcher checksum in a plaintext file.
    You can also hover over the link to see the checksum.
  • MD5:
    Local copy of the MD5 launcher checksum in a plaintext file.
    You can also hover over the link to see the checksum.
  • URL:
    Local copy of the remote repository HTTP URL in a plaintext file.
  • GRUJ:
    Local copy of a GRUJ bootloader for that specific launcher.
    GRUJ will download the launcher from the official repo.
    See http://gruj.org/
  • Opt:
    An optimized version of the launcher jar optimized with KZip & 7-Zip trials,
    deflate stream trimmed with DeflOpt & defluff.
    All the bytecode and resources are bytewise identical, but the resulting jar file is about 5% smaller.
